# 防火墙

## 概念解释

**防火墙**是网络安全设备，根据规则控制网络流量的进出。

就像公司的门卫：检查每个人的证件，决定谁能进入。

## 为什么重要

- **网络安全**：阻止未授权访问
- **流量控制**：管理网络带宽
- **合规要求**：满足安全标准

## 防火墙类型

### 1. 包过滤防火墙

检查IP包的头部信息。

**优点**：速度快
**缺点**：不能检查内容

### 2. 状态检测防火墙

跟踪连接状态。

**优点**：更安全
**缺点**：资源消耗大

### 3. 应用层防火墙

检查应用层数据。

**优点**：最安全
**缺点**：速度慢

## Linux 防火墙

### iptables

```bash
# 查看规则
iptables -L

# 允许SSH
iptables -A INPUT -p tcp --dport 22 -j ACCEPT

# 拒绝其他
iptables -A INPUT -j DROP
```

### firewalld

```bash
# 查看状态
firewall-cmd --state

# 开放端口
firewall-cmd --add-port=80/tcp --permanent

# 重载规则
firewall-cmd --reload
```

### nftables

```bash
# 新一代防火墙
nft list ruleset
```

## NAT（网络地址转换）

将私有IP转换为公网IP。

```bash
# IP伪装
iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
```

## 常见错误

1. **规则顺序**：规则按顺序匹配
2. **忘记保存**：重启后规则丢失
3. **过于宽松**：安全风险

## 实际应用

- **服务器**：保护服务安全
- **企业网络**：网络安全边界
- **家庭路由器**：基本防火墙

## 总结

防火墙是网络安全的第一道防线。理解防火墙类型和配置方法，有助于保护系统安全。
